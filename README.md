# Crux

A 3D printed hangboard that mounts above your door, with crimps, slopers, pockets, and pinches built in

Why I built it:

Wanted a way to train grip strength at home without dropping $200 on a store-bought board. Figured if I'm already printing and building everything else, why not this too — so I designed one from scratch.

What it does:

A 3D printed hangboard that mounts above your door with crimps, slopers, pockets, and pinches built in. All the holds bolt on so you can swap them out. Print it, mount it,train.
